2.Abstract Method : An Abstact class can have both abstract method which dont have body and non-abstract method that means that method have body.

class Program
{
    public abstract class CommonFeaturesOfMotorVehicles
    {
        public string? ModelNumber { get; set; }
        public string? CarColor { get; set; }
        public string? CarModel { get; set; }
        public string? BreakType { get; set; }
        public string? EngineType { get; set; }
        public string? FuelType { get; set; }

        //Non-Abstract Method
        public void SetData()
        {
            Console.WriteLine("Data Set");
        }
        //Abstract Method
        public abstract void GetData();//Declaring An abstarct Method
    };  
    class Car : CommonFeaturesOfMotorVehicles
    {
        public string? CarName { get; set; }
        public string? MusicSystem { get; set; }
        public string? AirConditioner { get; set; }

        public override void GetData() //Implementing Abstract Method
        {
            Console.WriteLine("Car Data Retrieved");
        }
    }
    class SuperCar : CommonFeaturesOfMotorVehicles
    {
        public string? CarName { get; set; }
        public string? MusicSystem { get; set; }
        public string? AirConditioner { get; set; }
        public string? NOS { get; set; }

        public override void GetData() //Implementing Abstract Method
        {
            Console.WriteLine("SuperCar Data Retrieved");
        }
    }
    public static void Main(string[] args)
    {
        Car car = new Car();//Use of Common feature Class
        car.SetData();//Use of Non-Abstract Method
        car.GetData();//Use of Abstract Method
    }
}

3.Sealed Class and Sealed Method
	1.Sealed Class: In C# when we don't want a class to be inherited by another class.we can declare the class as a sealed class.a sealed class cannot hav drived class.we use sealed keyword to create a sealed class.
class Program
{
    sealed class MySealedClass

            Console.WriteLine("Hello from MySealedClass!");
        }
    }

    class MyDerivedClass : MySealedClass // This will cause a compile-time error
    {
    }
    public static void Main(string[] args)
    {
        
    }

	2.Sealed Methods : During Method Overriding,if we don't want an overridden method to be further overridden by another class, we can declare it as a sealed method.
 class Program

Interface in C# : in C# an interface is similar to abstract class.however,unlike abstract classes all methods of an interface are fully abstract(methos without body).we use interface keyword to crate a interface.we cannot us access modifiers inside an interface.All members of an interface are public by default.an interface doesn't allow fields.Similar to abstract class ,interface helps us to achive abstraction in c#.interfaces provides specification that a class must follow.interfaces are use to achive multiple inheritance.
class Program
{
    interface IPolygon
    {
        //method without body
        void CalculateArea(int l, int b);
    }
    class Rectangle : IPolygon
    {
        //implementation of method
        public void CalculateArea(int l, int b)
        {
            Console.WriteLine("Area of Rectangle: " + (l * b));
        }
    }
    public static void Main(string[] args)
    {
        Rectangle r1 = new Rectangle();
        r1.CalculateArea(5, 10);
    }
}
