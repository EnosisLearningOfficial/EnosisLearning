*Microsoft SQL Server:
Microsoft SQL Server is a relational database management system developed by Microsoft in 1989. As a database server, it is a software product with the primary function of storing and retrieving data as requested by other software applications—which may run either on the same computer or on another computer across a network (including the Internet).
